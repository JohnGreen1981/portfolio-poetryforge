# PoetryForge

CLI toolkit for checking Russian syllabotonic meter and rhyme in LLM-generated poetry.

This is a sanitized portfolio repository scaffold. The source project already has a git repository, but this public version should be cleaned before publishing.

## What It Demonstrates

- Developer tool for validating LLM output.
- Meter analysis for Russian verse.
- Rhyme detection and rhyme search.
- Test-covered CLI workflow.
- Human + LLM + validator loop.

## Portfolio Role

This is the strongest code-quality case: it shows engineering discipline, tests and a clear verification use case.

## Status

Scaffold only. Code should be copied from the source repo after cleaning the working tree and deciding how to handle large data files.

## Before Publishing

- Clean dirty working tree in the source repo without losing local work.
- Decide which data files are safe and practical to publish.
- Exclude `.venv`, caches, local agent files and private notes.
- Run tests before publishing.
